#### 学びメモ

1. flexboxで中央揃えしたい + PC用の大きさを指定したいならmax-widthを使う
2. サイトの大要素にはIDを定義して、CSSにコメントと併記してIDを必ず記載してどこの要素をいじっているかわかりやすくする(classを多用しない)
3. SPやPCをベースに最小限のCSSの差分宣言だけで済ませるように意識したCSSを書く
4. divを使うのではなく、sectionやnavなどは意識して使う
